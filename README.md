# What is this?

Get shadows every time for non-designer.

# Installation

`npm i shadowwizard --save`

# Usage

import { shadowizard } from 'shadowizard'

shadowizard ({
    shadow_type: 'soft',
    padding: false
})

## Options

Supports 2 options;

* *shadow_type* - _hard / soft_ (Defaults to soft)
* *padding* - _boolean_ (Defaults to false)
